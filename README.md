# warden-layout-demo

这是基于warden-layout的一个demo

> demo演示地址：[https://demo.warden.vip](https://demo.warden.vip)

## 安装
yarn
```bash
$ yarn install
```

## 打包
yarn
```bash
$ yarn build
```

# 直接使用
如果需要在 build环境使用SettingDrawer组件，请开启.env中的配置

```ts
ENABLE_SETTING = true
```
🎈注意：需要禁用umi的mfsu
